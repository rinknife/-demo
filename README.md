# AI Shopping Agent - 智能购物比较助手

一个基于 大模型 + Function Calling + 多轮对话 的 AI 购物决策智能体 Demo
可自动理解用户需求 → 调用工具筛选商品 → 给出专业推荐

## 🎯 项目定位
面向电商场景的 AI 智能体（Agent）Demo，用于：
- 自动理解用户购物意图
- 自动筛选、对比、分析商品
- 提供性价比、店铺资质、价格走势等决策依据
- 实现类豆包的自然多轮对话

## 🧠 AI 智能体核心能力
1. 意图识别（品类、预算、品牌、偏好）
2. 函数调用（工具自动执行）
   - 按类别搜索商品
   - 按价格筛选
   - 按品牌筛选
   - 商品参数对比
   - 店铺资质校验
   - 性价比分析
3. 上下文记忆（多轮对话）
4. 结构化推荐输出

## 🔧 后续优化与改进方向
### 1. 对话体验优化：主动引导式交互（类 ChatGPT / 豆包）
当前版本为被动响应，仅在用户提问时回答。优化方向：
智能体在每次回答结束时主动提供可继续追问的方向，引导用户深入决策
例如：
“需要我帮你对比这几款的详细参数吗？”
“要我帮你查一下历史低价与购买时机吗？”
“需要我根据你的预算再筛选其他品牌吗？”
实现真正的对话式引导，提升用户停留时长与决策深度
### 2. 架构升级：全链路反馈闭环机制
在 Agent 工作流中加入可落地的反馈机制，让系统越用越准：
用户满意度反馈：每次推荐后记录用户是否满意
需求理解准确率反馈：记录用户意图是否被正确解析
工具调用效果反馈：记录筛选 / 对比 / 价格分析是否符合预期
长期偏好记忆：记录用户预算、品牌偏好、场景偏好
所有数据用于模型行为优化、提示词迭代、工具调度策略升级
### 3. 商品库与商店内容扩充（更真实电商环境）
当前仅为基础模拟数据，可进一步丰富：
扩充商品品类：耳机、平板、笔记本、家电、服饰、美妆等
增加真实电商维度：
店铺评分、月销量、好评率、退货率
商品标签：官方旗舰店 / 品牌店 / 专营店 / 第三方店铺
促销信息：满减、优惠券、限时价、套装价
商品评价：真实好评、中评、差评（重点突出风险点）
历史价格曲线：30 天 / 90 天价格波动
让 AI 决策更贴近真实电商购物场景
### 4. 智能体工作流深度重构（更像真正的 AI Agent）
拆分为更清晰的四层架构：
意图理解模块
任务规划模块（自动规划需要几步）
工具执行模块（并行调用、链式调用）
决策合成模块（汇总数据给出结论）
加入异常处理：信息不足自动追问、数据缺失自动提示
加入置信度判断：AI 知道自己 “不确定”，并告知用户
### 5. 体验增强：可解释推荐 + 避坑提醒
每条推荐增加可解释依据：为什么选这款、优先级是什么
自动识别高风险商品（低评分、多差评、非官方店）并提醒
输出结构化对比表，让用户一眼看懂差异
### 6. 扩展能力：未来可接入真实平台
对接京东 / 抖音商城开放平台
实时商品、实时价格、实时评价、实时店铺信息
从 Demo 升级为可正式上线的 AI 购物助手
